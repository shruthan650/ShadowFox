#  Calculator App (Java)

A Java-based calculator application built using Swing that supports both basic and advanced arithmetic operations through a graphical user interface.

---

##  Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### Advanced Operations
- Square root
- Square of a number

### UI
- Clean and simple interface
- Clear/reset functionality

---

##  Tech Stack
- Java
- Swing (GUI)

---

##  How to Run
cd src
javac *.java
java Main
